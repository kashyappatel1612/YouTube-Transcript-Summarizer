Summify is a modern **AI-powered web application** that converts YouTube videos into **clear, concise summaries** within seconds.

👉 Just paste a video link and get instant insights without watching the full video.

## ⚙️ How It Works
1 User enters YouTube video URL
2 Backend fetches transcript using API
3 Text is cleaned and preprocessed
4 NLP model generates summary
5 Summary is displayed on UI
